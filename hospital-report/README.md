# 入退院報告アプリ

LINE WORKS WOFF向けの入退院管理フォームアプリケーションです。

## 機能

- 利用者名の予測変換（マスタデータ連携）
- 医療機関の予測変換（エリア別）
- 入院/中止の条件分岐フォーム
- 診断名選択
- 契約終了処理とN列自動更新
- バリデーション機能

## 技術スタック

- Frontend: HTML5, CSS3, JavaScript (Vanilla)
- Backend: Google Apps Script
- Deploy: Netlify
